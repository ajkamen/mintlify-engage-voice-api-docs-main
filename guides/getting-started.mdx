# Getting started

Welcome to the Engage Voice API documentation. This guide will help you navigate the API and find what you need to get started.

## Common tasks

### Authentication
Before making any API calls, you'll need to authenticate. Choose the authentication method that matches your setup:

- **[RingCentral Authentication](/docs/authentication/auth-ringcentral)** - For RingCentral integrated accounts
- **[Engage Authentication](/docs/authentication/auth-engage)** - For standalone Engage Voice accounts
- **[Legacy Authentication](/docs/authentication/auth-legacy)** - For older implementations

### Making your first call
Once authenticated, here are the most common starting points:

#### Set up dialing campaigns
If you're setting up outbound calling:
1. **[Create a campaign](/docs/dialing/campaigns/campaigns)** - Set up your outbound dialing campaign
2. **[Configure dial groups](/docs/dialing/campaigns/dial-groups)** - Organize your dialing strategy
3. **[Import leads](/docs/dialing/leads/bulk-import)** - Add contacts to your campaign
4. **[Quick start guide](/docs/dialing/quick-start)** - Step-by-step dialing setup

#### Configure call routing
If you're setting up inbound call handling:
1. **[Queues overview](/docs/routing/queues/queues)** - Set up call queues
2. **[IVR configuration](/docs/routing/ivr/index)** - Create interactive voice response menus
3. **[Queue groups](/docs/routing/queues/queue-groups)** - Organize multiple queues
4. **[Quick start guide](/docs/routing/quick-start)** - Step-by-step routing setup

#### Manage agents
To set up and manage your agents:
1. **[Create agents](/docs/users/agents/agents)** - Add agents to your system
2. **[Agent groups](/docs/users/agents/agent-groups)** - Organize agents into teams
3. **[Skill profiles](/docs/users/agents/skill-profile)** - Configure agent skills for routing

### Monitor and analyze
Track your call center performance:
- **[Analytics overview](/docs/analytics/index)** - Access reporting and analytics
- **[Agent session reports](/docs/analytics/reports/agent-session-report)** - Monitor agent activity
- **[Call detail reports](/docs/analytics/reports/call-detail-report)** - Analyze call data
- **[Audit logs](/docs/analytics/audit-logs)** - Track system changes
